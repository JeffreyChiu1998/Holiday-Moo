# Holiday Moo~ 🐄

A React-based calendar application for managing trips and events.

## Features

- 📅 Interactive calendar view
- 🧳 Trip management with date ranges
- 📝 Event creation and editing
- 🏷️ Event categorization and tagging
- 📍 Location support with Google Maps integration
- 📄 Document attachments
- 💾 Import/Export calendar data
- 🔄 Undo/Redo functionality

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/JeffreyChiu1998/Holiday-Moo.git
cd Holiday-Moo
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Usage

- **Create Trips**: Use the Trip Panel to create and manage your trips
- **Add Events**: Click on time slots or use the Events Panel to create events
- **Associate Events with Trips**: Link events to specific trips for better organization
- **Export/Import**: Save your calendar data as JSON files for backup or sharing

## Built With

- React 18
- CSS3 with modern styling
- Local Storage for data persistence
